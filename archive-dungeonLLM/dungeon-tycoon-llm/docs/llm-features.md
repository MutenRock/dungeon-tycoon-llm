# LLM Features

## Autorisé
- narration de raid
- résumé quotidien
- dialogue de héros
- nommage procédural léger
- événements flavor

## Interdit
- calcul des dégâts
- validation du placement
- pathfinding
- économie
- vérité de l'état de jeu

## Règle
Le LLM enrichit l'expérience mais ne décide jamais de la logique centrale.
