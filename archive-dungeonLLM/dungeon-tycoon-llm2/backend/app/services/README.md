# Services

Couche de logique applicative.

## Services
- `dungeon_service.py` : placement, suppression, validation simple
- `pathfinding_service.py` : calcul de chemin dans le donjon
- `combat_service.py` : résolution déterministe minimale
- `economy_service.py` : ressources et coûts
- `raid_service.py` : orchestration d'un raid
- `event_service.py` : événements non-LLM
- `llm_service.py` : texte narratif uniquement
